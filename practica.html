Nueva práctica HTML

Mejorando habilidades en HTML, aprendiendo cómo usar diversas etiquetas para estructurar y dar formato al contenido de una página web. Este ejercicio está enfocado en dominar elementos básicos de HTML, como encabezados, párrafos, y etiquetas de texto, para mejorar la accesibilidad y la presentación de la información. A través de la práctica, se busca entender cómo se pueden aplicar estas etiquetas de manera efectiva en proyectos reales, creando páginas web más organizadas y visualmente atractivas.
